# Instagram Reel Virality Analyzer

## Project Overview

This project analyzes Instagram Reel virality using Python, SQL, and Power BI/Tableau.

The goal is to understand what makes Instagram Reels go viral by studying:

- Captions
- Hashtags
- Posting Time
- Engagement Patterns
- Audio Trends
- Reel Duration

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- TextBlob
- SQL (MySQL)
- Power BI / Tableau
- Jupyter Notebook

---

## Features

- Viral Caption Analysis
- Best Posting Time Detection
- Hashtag Effectiveness Analysis
- Engagement Rate Calculation
- Sentiment Analysis
- Dashboard Visualization
- Trend Tracking
